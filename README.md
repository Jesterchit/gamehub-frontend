# GameHub — Веб-платформа для покупки и управления играми

Учебный проект фронтенд-части веб-приложения для управления библиотекой цифровых игр.

## Описание

GameHub — это клиентское веб-приложение (SPA), которое позволяет пользователям:
- Регистрироваться и авторизовываться в системе.
- Просматривать каталог компьютерных игр с фильтрацией по жанрам, платформам и цене.
- Искать игры по названию.
- Просматривать детальную информацию об игре: описание, системные требования, разработчик.
- Управлять персональной библиотекой игр: добавлять, просматривать и удалять игры.

## Страницы приложения

| Страница | Описание |
|----------|----------|
| `index.html` | Вход и регистрация пользователя |
| `dashboard.html` | Личный кабинет со статистикой |
| `catalog.html` | Каталог игр с фильтрами и поиском |
| `game.html` | Детальная карточка игры |
| `library.html` | Персональная библиотека игр |

## Технологии

- HTML5 (семантическая вёрстка)
- CSS3 (Flexbox, Grid, медиазапросы, CSS-переменные)
- JavaScript (ES6+, работа с DOM, localStorage)
